<a name="creating-new-location"></a>
# Creating new location
Log in your account on [https://vrp.arvilab.com](https://vrp.arvilab.com) and switch to **Client Mode**: in the upper left corner of the page, from the ARVI VR drop-down menu, select **Client**.

<img src="{{ '/assets/img/client-mode.png' | relative_url }}" class="img-fluid mx-auto d-block" />

Select [**Locations**](https://vrp.arvilab.com/locations) section on the left panel -- this is a list of your locations. To create a new one, click **Add New**.

<a href="{{ '/assets/img/location-new.png' | relative_url }}" data-toggle="lightbox"><img src="{{ '/assets/img/location-new.png' | relative_url }}" class="img-fluid mx-auto d-block" style="max-width:700px;width:100%;" /></a>

Fill all required fields and click **Submit** to create a location.

<a name="registering-admin-client"></a>
# Registering Admin Panel and Client PC
To link computers to this location, go to the [**Downloads**](https://vrp.arvilab.com/downloads) section, download and install **Admin Panel Installer** on the computer from which you will launch and control the game sessions, and **Client Installer** on the gaming computers that will run the games.

<a href="{{ '/assets/img/downloads.png' | relative_url }}" data-toggle="lightbox"><img src="{{ '/assets/img/downloads.png' | relative_url }}" class="img-fluid mx-auto d-block" style="max-width:700px;width:100%;" /></a>

At the License Activation stage, select your location from the drop-down list and give the name of the workstation. It will be displayed under this name in the Admin Panel.

<a href="{{ '/assets/img/workstation-registration.png' | relative_url }}" data-toggle="lightbox"><img src="{{ '/assets/img/workstation-registration.png' | relative_url }}" class="img-fluid mx-auto d-block" style="max-width:500px;width:100%;" /></a>

Follow the installation procedure for all computers that need to be in this location.
